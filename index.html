Biaya Layanan

Panduan Cepat

  1. Hubungkan Phantom Wallet.
  2. Pilih jaringan (Devnet untuk uji coba, Mainnet untuk nyata).
  3. Isi informasi token dasar (nama, simbol, desimal, suplai).
  4. Untuk metadata: Upload metadata JSON ke IPFS (Pinata), salin CID, dan isi di field "Metadata JSON CID".
  5. Klik "Buat Token dengan Metadata".
  6. Konfirmasi transaksi di wallet Anda (3 transaksi: create mint, create metadata, mint supply).
  7. Biaya layanan 0.15 SOL akan otomatis dipotong per transaksi.
  8. Token akan muncul di wallet Phantom Anda dalam 1-2 menit.
